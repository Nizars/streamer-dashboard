Get Twitch User Data

Bellow is a method to consume user data from Twitch's API

RESPONSE

  • Display Name
    "user.display_name"
  • Broadcaster Type
    "user.broadcaster_type"
  • Description
    "user.description"
  • User ID
    "user.id"
  • User Type
    "user.type"
  • View Count
    "user.view_count"
  • Profile Image
  • Offline Image
